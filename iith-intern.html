Fin Motion Optimization — Bio-Inspired Underwater Robot

Python XGBoost Raspberry Pi Servo Control scikit-learn

Overview

Developed an ML-based fin motion optimization system for a bio-inspired underwater robot that mimics fish locomotion using two servo motors per fin. The goal was to find the optimal combination of servo frequency, amplitude, and phase offset that maximizes thrust while minimizing energy consumption — without spending months on hardware trials.

Problem

Testing all possible fin parameter combinations physically in a water tank would require thousands of trials and months of work. A smarter, data-driven approach was needed.

What I Did

  • Mounted the robot on a force-measurement rig in a water tank and programmed Raspberry Pi to send sinusoidal commands to two servo motors per fin
  • Ran 270 controlled experiments across 3 motion modes, repeating each 3 times to handle sensor noise — generating 810 clean training datapoints
  • Trained an XGBoost regression model to predict thrust and power for any servo parameter combination, achieving R² of 0.96
  • Performed a 16,000-combination grid search on the trained model in under 3 seconds — replacing months of physical testing
  • Validated top predictions in the tank, confirmed within 5% error, and deployed optimized parameters as a real-time lookup table on Raspberry Pi

Key Finding

A phase offset of 85–90° between the two servo motors consistently produced peak efficiency across all motion modes — independently confirming the bio-inspired principle of how real fish swim.

Results

  • R² of 0.96 on thrust and energy prediction
  • 78% reduction in hardware testing time
  • Model predictions validated within 5% error on real hardware
  • Successfully deployed across 6 motion modes on Raspberry Pi onboard controller
